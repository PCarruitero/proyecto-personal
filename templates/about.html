Warframe Wiki

¿Qué es Warframe?

Warframe es un videojuego de acción cooperativo en tercera persona desarrollado por Digital Extremes. Los jugadores controlan Warframes, exotrajes biomecánicos con habilidades únicas, y exploran el sistema Origin enfrentándose a distintas facciones.

¿Qué es Warframe Wiki?

Warframe Wiki es un proyecto web educativo que funciona como una base de datos visual de Warframes. Permite explorar personajes, estadísticas y habilidades de forma clara, ordenada y accesible.

Objetivo del proyecto

El objetivo de este proyecto es aplicar conocimientos de desarrollo web frontend y backend, integrando bases de datos y buenas prácticas de diseño, creando una aplicación funcional inspirada en un videojuego real.
